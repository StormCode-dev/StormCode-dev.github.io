# To setup:

1. Place the EXE in any folder you would like
2. Copy the path of the folder
3. Go to your environment variables, and under system, paste this
4. Restart your IDE, Shell, or system


# Getting started:

Run "amethyst debug" to verify your install

If all goes well, you should see:
```
AVCS: Debug Status------------------------------------------------------------------
AVCS: Build 9-23-2025 : STABLE
AVCS: Current Directory: C:\Users\micro
AVCS: ADE version: 0.1a (Dark Tides)
AVCS: Have any issues? Report them on either the GitHub repo or my Discord server.
```

To get started, you can run "amethyst init" when cd'd into a directory, or manually specify.

1: amethyst init (assumes cwd)

2: amethyst init -p "D:\path\torepo"

**It's important to note that Windows Defender, or any AV Product will flag Amethyst as malicious because Amethyst, as of right now, has zero certificates, like a Norton Cetificate (they are 200 dollars). You would have to make a rule on your AV for Amethyst to run properly.
