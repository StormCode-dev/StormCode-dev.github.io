# CompressAll

```
#compresses everything amethyst made itself (exluding this file!) and is useful to save space.
#if marked true, you cant read anything normally. defaults to false.
compressAll=false
```

This takes a Git approach to how data is stored. Data is compressed to be more compact, and can help with space. This option has not been fully implemented yet. Mark as true in order to employ compression. It should be noted that when you change this and run a command, it will take longer to chew through your files, since an extra step to interpret will be needed.
