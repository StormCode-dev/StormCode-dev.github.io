# Commands

Amethyst has commands that let you do who knows what with your project. The current list for Amethyst 

1. init: initialized amethyst in a specified directory
2. geoupd: updates the geode files with the latest changes
3. geopush: pushes the changes to a .geopush file
4. validate: ensures the installation works, repairs if needed
5. uninstall: removes amethyst from a specified directory. wipes your data, be careful!
6. debug: default command. provides stats on your current installation. useful for, as the name suggests, debugging
7. reinit: uninstalls amethyst from a specified directory, then reinstalls it back there. wipes your data, be careful!


This is a superficial overview. Check more of the commands' official pages to see more.
