# Debug

This takes no arguments, and is useful to ensure your version of Amethyst is up to date.

Usage includes:

```
amethyst debug
```

and an expected output could be

```
AVCS: Debug Status------------------------------------------------------------------
AVCS: Build M-DD-YYYY : STATION/BRANCH
AVCS: Current Directory: C:\path\torepo
AVCS: ADE version: 0.1a (Dark Tides)
AVCS: Have any issues? Report them on either the GitHub repo or my Discord server.

```
